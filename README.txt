
nom du projet : bloc_formulaire_de_connection

descriptif    : ce projet a pour but de réaliser un bloc html contenant un formlulaire de connexion.
		il contient 2 champs : mail, mot de passe et un bouton pour valider
		il contient egalement deux liens : un pour "s'inscrire", un pour "mot de passe oublié"

	        Ce bloc formulaire peut être développé en tant que sous partie d'un site internet, dans le cadre d'un travail en équipe.
                Ce projet est un cas d'école pour montrer et utiliser git et github.

developpeur   : JOUY jean-christophe, peudo : codyce sur github  

date debut    : 18/08/2016

detail tech.  : Le formulaire serra réalisé avec des balises "imput" en html et affiné par un fichier en CSS
		Le traitement du login et mot de passe est assuré par une methode POST vers un fichier PHP ( coté serveur )
                Ce projet dans le cas de notre etude comportera 4 fichiers
                - une fichier README.txt
                - un fichier index.html
                - un fichier style.css
                - un fichier sinscrire.php
         
                Ce projet au moment de l'envoi pour correction est volontairement inachevé ( il manque le fichier "motdepasseoublie.php")
                afin, qu'il puisse servir de base pouvant être continué ultérieurement par un autre développeur.
